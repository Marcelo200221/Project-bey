# Backend de Proyecto-Bey

Backend desarrollado en Django y Django REST Framework para la gestión de usuarios y lógica del sistema.

## Librerías utilizadas

1. dj_rest_auth - Autenticación y registro de usuarios.

2. django-rest-framework - Creación de API REST.

3. rest_framework_simplejwt - Autenticación mediante JWT.

4. django-allauth - Gestión avanzada de autenticación.

5. django-cors-headers - Configuración de CORS para conexión con el frontend.

6. mysqlclient - Conector para bases de datos MySQL.
